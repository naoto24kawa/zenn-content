個人開発で死活監視+Cron監視+SSL監視+ページ変更検知を全部カバーしようとしたら、月$33かかりました。しかも MCP 対応はゼロ。Claude に「サーバーの状態は?」と聞いても何も返ってこない。

2026年にもなって、監視データを見るために毎回ブラウザでダッシュボードを開く必要がある。これが監視ツール市場の現実です。

自分が探していたのは:

- 死活監視、Cron 監視、SSL 監視、ページ変更検知が **1つで** できる
- **AI エージェントから操作できる** -- 「サーバーの状態は?」で答えが返ってくる
- 個人開発者が払える価格

結論から言うと、全部満たすツールはありませんでした。でも各ツールにはそれぞれ良いところがあったので、正直にレビューします。

## UptimeRobot

**死活監視の定番。無料枠が強い。**

### 良い点
- 無料で 50 モニターまで使える。個人開発なら十分
- セットアップが簡単。URL を入れるだけで始められる
- ステータスページが無料で作れる

### 悪い点
- UI が全部英語。設定項目が多くて迷う
- 無料プランは 5分間隔。気づいた頃にはダウンして数分経ってる
- Cron 監視やページ変更検知はない。死活監視に特化
- API はあるが、MCP 非対応。AI エージェントから「今の状態は?」と聞くことができない
- LINE 通知は非対応

**向いてる人**: 死活監視だけでいい、英語 UI が気にならない人

## Better Stack

**モダンな UI でインシデント管理まで一体化。**

### 良い点
- UI がきれい。ダッシュボードの視認性が高い
- Uptime + ログ + インシデント管理が 1つのプラットフォーム
- ステータスページのデザインが洗練されている

### 悪い点
- 無料は 5 モニターまで。個人開発で複数サービス持ってるとすぐ足りない
- Team プランは $21/月。個人で払うにはちょっと高い
- 日本語対応なし。ドキュメントも英語のみ
- 機能が豊富すぎて、個人開発にはオーバースペック感がある。使いこなせてない機能に毎月払ってる感じがする
- ページ変更検知がない。「競合のページが変わったら教えて」は別ツールが必要

**向いてる人**: チーム開発でインシデント管理もまとめたい人

## Healthchecks.io

**Cron 監視に特化。シンプルで好き。**

### 良い点
- Cron ジョブの監視はこれが一番わかりやすい
- ping を送るだけのシンプルな設計。実装が楽
- 無料で 20 チェックまで。OSS 版もある
- UI が素朴で迷わない

### 悪い点
- Cron 監視に特化しすぎていて、HTTP の死活監視はおまけ程度
- SSL 監視やページ変更検知はない。結局他のツールも必要になる
- 日本語対応なし
- 通知先は多いが、API や MCP での外部連携は想定されていない
- ping URL を自分のコードに埋め込む必要がある。既存のバッチに1行足すだけとはいえ、触りたくないスクリプトってあるよね

**向いてる人**: Cron ジョブの監視だけしたい人

## Visualping

**Web ページの変更検知に特化。**

### 良い点
- ページの見た目の変化をスクリーンショットで比較してくれる
- 競合サイトの監視や、官公庁ページの更新チェックに便利
- 検知精度の調整ができる

### 悪い点
- 無料は 5 ページまで、チェック間隔も長い
- 月 $10 は変更検知だけと考えると高め
- 死活監視や Cron 監視はできない。完全に別カテゴリ
- 日本語対応なし
- 「ここが変わったら教えて」の設定が結構面倒。DOM セレクタの指定とか、ノイズ除外のチューニングとか

**向いてる人**: 特定ページの変更をトラッキングしたい人

UptimeRobot ($7) + Healthchecks ($16) + Visualping ($10) の3つで最低 $33/月。Better Stack も足すと $54/月。個人開発にはキツい。Manako なら Pro $5/月から始められる。

そして全ツール共通の問題として、**どれも AI エージェント連携を想定していない**。2026年にもなって、監視データを取得するのに毎回ブラウザでダッシュボードを開く必要がある。Claude や ChatGPT に「今のサーバーの状態は?」と聞いて答えが返ってくるツールが1つもない。MCP 対応しているサービスはゼロ。

## まとめ

- 死活監視だけなら UptimeRobot の無料枠で十分
- Cron 監視なら Healthchecks.io が一番シンプル
- インシデント管理込みなら Better Stack
- 変更検知なら Visualping
- 全部まとめたい + AI エージェント連携がほしいなら...まだ選択肢がない
